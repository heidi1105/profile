WHERE R U

WHERE R U is an online clock-in web app allowing employers to keep track of the location of the employees in the assigned shift. After signing in, the system would direct employers and employees to different pages. An employer could assign shift of a specific location for employees. When employees login, they need to click button for GPS to track their location. After that, employees could sign in and sign out their shift.

TECHNOLOGY
  • Language : Java
  • Framework: Java Spring Boot
  • Database : MySql
  • Frontend: JSP, CSS, Bootstrap
  • API: GoogleMap Geolocation
ROLES

In this project, I was responsible for front-end design, Database models, and the JSON integration while the other teammate was responsible for the Geolocation API and queries to show previous and current shifts.

SLASH

SLASH is a comprehensive global e-commerce site selling exclusive products. LovFit

LovFit is an online dating site which aims to help atheltic peole find people with similar interest by the use the STRAVA, a social media to track hiking, running, biking and more. Users require a Facebook Account to verify their identity and a Strava account to find their partners. When users login, they could view the Strava profiles of all the other users who have signed up for LovFit. If they like it, they could LIKE the person.

TECHNOLOGY
  • Language : Python
  • Framework: Django
  • Database : SQLite
  • Frontend: HTML5, CSS
  • API: Facebook API, Strava API (OAuth)
ROLES

LovFit was built in a close cooperation with two other partners with multiple discussion in the matching logic and database design in integrating Facebook and Strava. I was responsible for the Facebook Login, all the database queries and part of the front-end design.
